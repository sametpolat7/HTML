The Anchor Element

The text <a> HTML element (or anchor element), with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.

Content within each <a> should indicate the link's destination. If the href attribute is present, pressing the enter key while focused on the <a> element will activate it.

rel tag

The rel tag is a microformat data method that makes the page link meaningful to search engine crawlers. rel="noopener no referrer" The primary purpose here is security; When you click on links with target="_blank" attribute on your web page, the link will open a new browser tab. This new page that opens may provide access to some information about the source page that opens it, or there may be a risk of controlling it, albeit partially. This poses a potential potential vulnerability risk.

The noopener command tells the browser to reset all the variables of the source page that opened the page opened in the new tab.

Noreferrer gives the command to reset the referer value of the page opened in the new tab; so the new page will not be able to access the address data of the source page it came from.
