# The HEAD Tag
The HTML `<head>` element is a container for the following elements: `<title>`, `<style>`, `<meta>`, `<link>`, `<script>`, and `<base>`.

## Contents
- Title
- Style
- Meta
    - meta charset
    - meta name="keywords"
    - meta name="description"
    - meta name="viewport"
    - meta name="author"
    - meta http-equiv="refresh"
- Language
- Link
- Base
